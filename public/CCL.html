About Cmazey Command Line

Cmazey Command Line is a project that I started back in 2022 during my freshman year of high school. This project includes a lot of projects that I have worked on since high school. The Node command line is a custom command line with my own commands that I made while I was learning Javascript. The py command line is a custom command line that I made in early 2023 while I was learning Python. Cmazey Calculator, which is a terminal calculator that I made for fun at first, has turned into an AP project for my junior year. Cmazey Command Line also includes smaller projects that are stored in others, and it also comes with lessons that store the lessons that I had to do during my freshman and junior years.
